프로젝트 참여 개요

처음 진행한 BO 프로젝트로 전반적인 작업을 진행하였습니다.

주요 역할

  • 기획서 및 디자인 분석
    • 기획서와 디자인 검토 후 질의 사항을 구글 프레젠테이션으로 정리하여 전달하고 피드백을 받으며 타 팀과 소통
  • 퍼블리싱 작업 수행
    • form, button, title, table, tooltip, tab, layer popup 공통 컴포넌트 작업
    • PM님의 지시 하에 전체 페이지 작업 진행
    • 페이지 작업이 완료되면 기획서, 디자인을 검토하여 1차적인 퍼블 코드 검수와 브라우저 테스트 진행
    • 백엔드 개발팀에 전달할 작업 페이지 목록 메일 양식 리스트업

성과 및 배움

  • IntersectionObserver API를 처음 사용해 보았습니다. 요소가 뷰 포트에 관찰됐느냐의 여부를 체크할 수 있어 사용성이 높을 것 같다는 생각이 들었습니다. 다음 프로젝트 작업 시에도 사용 가능할 것으로 예상합니다.
